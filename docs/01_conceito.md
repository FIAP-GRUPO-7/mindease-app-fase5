PT-BR:
# MindEase — Conceito Central

O MindEase é um aplicativo de acessibilidade cognitiva focado em reduzir a fadiga decisória.

## Princípio Central
O sistema assume decisões sobre interface, ritmo e foco, permitindo que o usuário se concentre apenas no que importa no momento.

## O que o MindEase é
- Um ambiente de execução guiada
- Um aplicativo orientado por comportamento
- Uma ferramenta de suporte cognitivo

## O que o MindEase não é
- Um site institucional
- Um painel de produtividade
- Um gerenciador de tarefas com muitos recursos

## Critérios de Sucesso
Se o usuário pensar menos e progredir com calma, o sistema está funcionando.


EN:
# MindEase — Core Concept

MindEase is a cognitive accessibility application focused on reducing decision fatigue.

## Core Principle
The system assumes interface, rhythm, and focus decisions so the user can concentrate only on what matters now.

## What MindEase Is
- A guided execution environment
- A behavior-driven application
- A cognitive support tool

## What MindEase Is Not
- An institutional website
- A productivity dashboard
- A feature-heavy task manager

## Success Criteria
If the user thinks less and progresses calmly, the system is working.
